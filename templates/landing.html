Get Started

Install SpaceBridge for Claude Code

Instantly connect your Claude Code instance to GitHub, GitLab, and Jira with our streamlined setup script. Copy and paste the following command in your terminal:

curl -sSL https://install.spacebridge.io/claude | bash -s -- --auth-key YOUR_AUTH_KEY

The script will automatically:

  • Install the SpaceBridge MCP server
  • Set up your tracker integrations with Claude Code
  • Configure your issue tracking environment

Install SpaceBridge for Cursor

Connect your Cursor IDE to GitHub, GitLab, and Jira issue trackers with our one-line installer. Copy and run this command in your terminal:

curl -sSL https://install.spacebridge.io/cursor | bash -s -- --config cursor-default

This setup provides:

  • Automated issue tracking for Cursor AI
  • Cross-repository issue search capabilities
  • Smart duplicate issue detection

Install SpaceBridge for Windsurf

Supercharge your Windsurf IDE with unified issue tracking across GitHub, GitLab, and Jira. Copy this command to your terminal:

curl -sSL https://install.spacebridge.io/windsurf | bash -s -- --integrate-agentic

The Windsurf integration includes:

  • Seamless issue tracking for agentic workflows
  • Automatic context augmentation for your LLMs
  • Enhanced similarity search across all your trackers
