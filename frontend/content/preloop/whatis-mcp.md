# What is MCP?

The **Model Context Protocol (MCP)** is an open standard that enables AI assistants to securely connect to external data sources and tools.

## How Preloop Uses MCP

Preloop AI acts as an MCP server that provides:

- **Human-in-the-loop approval** - Pause AI actions for human review before execution
- **Tool orchestration** - Connect multiple MCP servers through a unified interface
- **Audit logging** - Track all AI tool invocations and approvals

## Getting Started

1. Generate an API key in **Settings > API Keys**
2. Configure your AI assistant to connect to your Preloop instance
3. Start using AI tools with human oversight
